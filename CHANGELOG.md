# Changelog

All notable changes to Quantum Workspace are documented here.

## [1.0.0] - 2026-04-02

### Added
- **Quantum Brain Engine** - Unified AI engine with context, memory, cache, and automation
- **Simple Mode** - Non-technical user interface with goal-based project creation
- **Developer Mode** - Full-featured developer experience
- **AI Agents** - Architect, Developer, and Designer agents for full SDLC
- **One-Click Build** - Auto-detecting build system for any project type
- **One-Click Deploy** - Deploy to Vercel, Cloudflare Pages, or Netlify
- **Visual Git** - Full visual Git interface with staging, commits, and branches
- **AI Terminal** - Terminal with smart command suggestions
- **Project Templates** - 12 pre-built templates (React, Next.js, Vue, etc.)
- **Plugin System** - Extensible plugin framework
- **CLI Tool** - Command-line interface with Git and MCP support
- **Skills System** - Business viability, requirements, system design skills
- **GitHub Actions CI** - Automated build and release workflows
- **Full Documentation** - Complete docs for all features

### Changed
- License changed from MIT (VSCodium) to PROPRIETARY
- Brand changed from VSCodium to Quantum Workspace
- Updated package.json with Quantum branding

### Removed
- VSCodium-specific telemetry
- Microsoft telemetry references
- Legacy VSCodium documentation
