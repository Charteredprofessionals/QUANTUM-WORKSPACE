# Quantum Workspace - Command Reference

## All Commands

### Mode Commands
| Command | Description |
|---------|-------------|
| `quantumWorkspace.start` | Launch Quantum Workspace |
| `quantumWorkspace.simpleMode` | Switch to Simple Mode (non-technical) |
| `quantumWorkspace.developerMode` | Switch to Developer Mode |
| `quantumWorkspace.switchMode` | Toggle between modes |

### Build Commands
| Command | Description |
|---------|-------------|
| `quantumWorkspace.build` | Full build with auto-detection |
| `quantumWorkspace.quickBuild` | Fast build with progress |
| `quantumWorkspace.clean` | Remove build artifacts |
| `quantumWorkspace.package` | Package for distribution (ZIP/TAR) |
| `quantumWorkspace.buildStatus` | Show build status |

### Deploy Commands
| Command | Description |
|---------|-------------|
| `quantumWorkspace.deploy` | Deploy (choose provider) |
| `quantumWorkspace.deployVercel` | Deploy to Vercel |
| `quantumWorkspace.deployCloudflare` | Deploy to Cloudflare |
| `quantumWorkspace.openDeployment` | Open deployment URL |

### Git Commands
| Command | Description |
|---------|-------------|
| `quantumWorkspace.git` | Open Visual Git panel |
| `quantumWorkspace.gitStatus` | Show Git status |

### Terminal Commands
| Command | Description |
|---------|-------------|
| `quantumWorkspace.terminal` | Open AI Terminal |
| `quantumWorkspace.terminalSuggest` | Get AI suggestions |

### Template Commands
| Command | Description |
|---------|-------------|
| `quantumWorkspace.templates` | Browse project templates |
| `quantumWorkspace.createProject` | Create from template |

### Plugin Commands
| Command | Description |
|---------|-------------|
| `quantumWorkspace.plugins` | Manage plugins |
| `quantumWorkspace.createPlugin` | Create new plugin |

### AI Brain Commands
| Command | Description |
|---------|-------------|
| `quantumWorkspace.indexCodebase` | Index project for AI |
| `quantumWorkspace.learnProject` | Learn project structure |

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+P` | Command Palette |
| `Ctrl+Shift+G` | Visual Git |
| `Ctrl+Shift+T` | AI Terminal |
| `Ctrl+Shift+D` | Deploy |
